Kotlin学习笔记——GridView

Kotlin学习笔记——GridView

使用方法

  1. 设计好界面
  2. 新建一个小毛驴文件，这个小毛驴文件是GridView中，每一个Item的界面布局文件
  3. （可选）编写一个数据类，用来保存每个item中的数据，用data class可以很方便
  4. 编写一个继承BaseAdapter适配器的类
  5. 如果编写了数据类（起了一个c++中结构体的作用，因为数组只能传递一个），创建对应的List并且赋值
  6. 给GridView添加适配器
